# Data Governance

Reference data is part of the trust model. It must never become an unreviewed dumping ground.

Every dataset must record:

- dataset ID;
- source title and URL or publication details;
- license or public-domain status;
- generation or extraction method;
- checksum;
- units;
- uncertainty, if known;
- applicable model and valid range;
- reviewer sign-off.

Datasets with unclear or restricted redistribution rights must not be included in distributable core crates.
