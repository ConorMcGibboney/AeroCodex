# Roadmap

## v0.0.0 Founder Baseline

Project charter, name, repository plan, dependency policy, evidence-level definitions, initial roadmap, risk register, founder pack, and deployable repository foundation.

Workspace scaffold; core, units, validation; typed quantities; error model; `EngineeringResult`; Codex Card schema; first generated report.

Exit gate: reference-table reproduction report generated.

Isentropic flow, area-Mach relation, choked flow, normal shock, oblique shock, Prandtl-Meyer functions, branch-explicit inverse solvers.

Exit gate: reference tables, inverse tests, limit tests, and branch tests pass.
