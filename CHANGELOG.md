# Changelog

All notable changes to AeroCodex should be documented here.

The project uses Cargo-compatible semantic versioning for packages and the separate AeroCodex project phase label during early development. Phase `0.001` maps to Cargo version `0.0.1`.

## [0.0.1 / Phase 0.001] - 2026-06-14

### Added

- Phase 0.001 versioning policy and milestone train.
- Starter Rust calculations across atmosphere, thermodynamics, gas dynamics, aerodynamics, propulsion, heat transfer, structures, flight dynamics, astrodynamics, and bio-regenerative life support.
- Evidence cards for each Phase 0.001 starter calculation.
- Source registry seed records for compressible flow, U.S. Standard Atmosphere, NASA CEA comparison, NASA life support BVAD research, JPL astrodynamic parameters, and NAIF frame references.
- Agentic Rust scaffold for registry, schemas, retrieval, companion vector tokens, tool invocation, trace/replay, local server, and CLI.
- Expanded roadmap through v1.0 and beyond.

### Caveats

- Phase 0.001 is not a production engineering release.
- Source review and reference validation are pending for many starter equations.
- The agentic layer is scaffold/planning code, not a production tool server.

## [0.0.0] - 2026-06-14

### Added

- Founder baseline repository files.
- Dual MIT OR Apache-2.0 licensing.
- Initial 100% Rust no-wrapper dependency policy.
- Initial validation-card schema and example evidence card.
- Minimal Rust workspace with core, units, constants, numerics, validation, data, and gas-dynamics starter crates.
- GitHub Actions CI skeleton.
